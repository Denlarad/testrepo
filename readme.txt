1) git init - инициализация git проекта, делается один раз. Для начала работы с git

2) git status - посмотреть состояние проекта, были ли добавлены файлы, были ли изменения, есть ли файлы которые нужно закомитить.
3) git add * - добавить фалы в git, иначе они не будут закомичены
4) git commit -m 'description' - зафиксировать изменения
5) git remote add origin https://github.com/Mrden31/ИМЯ_РЕПОЗИТОРИЯ.git - если еще не добавлял ссылку на репозиторий.
5) git push -u orign master - отправить изменения на сервер, после можно просто отправлять git push
